# ReadIt


## Description
ReadIt is a free and open source reader webapp made using Vue.js
## Features
- Read text and HTML on a unified clean interface
- HTML Sanitization
- Query Selector, Query Selector All, Separators
- Themes

## Production Build
[Here](https://readit.vercel.app/)

```
yarn install
```
